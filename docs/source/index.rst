
UNICORE File Transfer (UFTP)
============================

The UFTP contains server and client components for the `UNICORE <https://www.unicore.eu>`_ high-performance file transfer toolkit:

* :doc:`UFTPD server <uftpd>` - contains the core Java libraries as well as a reference implementation of the 'uftpd' server
* :doc:`PyUFTPD server <pyuftpd>` - contains the production version of the 'uftpd' server, written in Python
* :doc:`Auth server <authserver>` - a set of services providing authentication for UFTP as well as data sharing features
* :doc:`UFTP client <uftp-client>` - the standalone "uftp" client application
* :doc:`Datashare <datashare>` - library containing support code for the data sharing feature
