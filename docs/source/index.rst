UNICORE File Transfer (UFTP)
****************************
Last change: |today|

The UFTP contains server and client components for the `UNICORE <https://www.unicore.eu>`_ high-performance file transfer toolkit:

* :ref:`uftpd` - contains the core Java libraries as well as a reference implementation of the 'uftpd' server
* :ref:`pyuftpd` - contains the production version of the 'uftpd' server, written in Python
* :ref:`authserver` - a set of services providing authentication for UFTP as well as data sharing features
* :ref:`uftp-client` - the standalone "uftp" client application
* :ref:`datashare`- library containing support code for the data sharing feature
